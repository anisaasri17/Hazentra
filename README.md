# Hazentra

## Android Mobile Application for Hazard Reporting and Mapping

Hazentra is an Android-based mobile application developed for the ICT602 Mobile Technology Group Project. The application applies the concept of crowdsourcing by allowing users to report and view location-based hazards through Google Maps. Hazard reports are stored on a PHP and MySQL server and displayed dynamically within the mobile application.


## Project Overview

Hazentra enables users to identify and report hazards that may affect public safety. The application displays hazard reports on an interactive Google Map and allows users to submit new reports using their current GPS location. All submitted reports are stored on a server and managed through a web-based administrator dashboard.


## Features

- Display current GPS location
- Display dynamic hazard markers from the server
- Submit hazard reports with GPS coordinates
- View hazard reports
- View hazard statistics
- Light and Dark theme support
- About page
- Web-based administrator dashboard


## Hazard Categories

The application supports the following hazard categories:

- Road Hazard
- Environmental Hazard
- Building Hazard
